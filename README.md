# 📒 Contact Book

A simple command-line Contact Book application built using Python.

## 📌 About the Project

This project allows users to manage their contacts through a menu-driven interface. Users can add, view, search, and delete contacts directly from the terminal.

## ✨ Features

* ➕ Add a new contact
* 📋 View all contacts
* 🔍 Search contacts by name
* 🗑️ Delete a contact
* 🚪 Exit the application

## 🛠️ Technologies Used

* Python 3

## 📂 Project Structure

```text
main.py
README.md
```

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python main.py
```
